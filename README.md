# Веб-приложение "Weather Forcast"

### Описание:

Веб-приложение для просмотра прогноза погоды в любой точке мира на ближайшие 7 дней.

_**Особенности:**_

- Адаптивный интерфейс
- Сохранение последней локации
- Технология PWA

---

### Используемые технологии:

> - **Figma**
> - **HTML**
> - **SCSS**
> - **TypeScript**
> - **Vue**
> - **Vite**
